THE BEATLES QUIZ

Built with Javascript, jQuery, Css and Sass

It is an interactive quiz based on and inspired by my love for The Beatles. It has starter page, a question, answer, and score counter that updates with each question, a feedback page for each question and a restart button at the end. It is responsive and a11y friendly.
